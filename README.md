О проекте

Приложение для статистического анализа данных с прогнозированием методом скользящей средней. Реализовано тремя участниками команды, каждый из которых выполнил свой вариант задания в отдельной ветке.

Выполненные задания

Загрузка JSON файла с данными об инфляции за каждый год. Отображение в табличном формате. Графики зависимости параметров от года. Прогноз методом скользящей средней на N лет вперёд с визуализацией на графике.
	Вариант 10 — Данные об инфляции России (Голубев Р.Д.)

Загрузка JSON файла с данными о населении за каждый год. Отображение в табличном формате. Графики зависимости параметров от года. Прогноз методом скользящей средней на N лет вперёд с визуализацией на графике.

Как запустить

    Клонировать репозиторий:
https://github.com/52678vasya-jpg/Laba3/tree/main

    Открыть InflationAnalyzer2007.sln в Visual Studio 2022
    Убедиться что установлен .NET 8
    Запустить через F5 или Debug → Start

При запуске выбрать нужный модуль в главном меню и загрузить соответствующий файл с данными.
